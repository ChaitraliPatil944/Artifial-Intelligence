# 2. What is a Biological Neuron?

A neuron is a specialized nerve cell that receives, processes, and
transmits information using electrical and chemical signals.

The human brain contains roughly **86 billion neurons**, each connected
to thousands of others.

# 4. Structure of a Neuron

``` text
        Dendrites
      \   |   /
       \  |  /
        \ | /
      [ Cell Body ]
            |
         Axon
            |
     Axon Terminals
            |
        Synapse
            |
     Next Neuron
```

### Dendrites

Receive signals from other neurons.

### Cell Body (Soma)

Processes incoming information.

### Axon

Carries the electrical signal away from the cell body.

### Axon Terminals

Pass signals to other neurons.

### Synapse

Tiny gap where communication occurs using neurotransmitters.

# 5. How a Neuron Works

``` text
Input Signals
      │
Dendrites
      │
Cell Body
      │
Decision:
Fire?
      │
     Yes
      │
Axon
      │
Next Neuron
```

If the incoming signal exceeds a threshold, the neuron "fires."

------------------------------------------------------------------------

# 6. Communication Between Neurons

1.  A neuron receives electrical signals.
2.  It combines the signals.
3.  If strong enough, it generates an action potential.
4.  Neurotransmitters cross the synapse.
5.  The next neuron receives the signal.

``` text
Neuron A
   │
 Action Potential
   │
 Synapse
   │
Neuron B
```

# 8. Advantages of Brain-inspired Computing

-   Learns from examples.
-   Highly parallel.
-   Fault tolerant.
-   Adapts over time.

These ideas influenced modern neural network design.

------------------------------------------------------------------------

# 9. Limitations

Artificial neurons are **extremely simplified** compared to real
neurons.

Real brains involve:

-   Complex chemistry
-   Memory mechanisms
-   Timing effects
-   Plasticity

Deep learning captures only a small part of this complexity.

# 11. Interview Questions

### Why study biological neurons in AI?

Because artificial neural networks were inspired by how biological
neurons process information.

### What are the main parts of a biological neuron?

Dendrites, cell body, axon, axon terminals, and synapse.

### What is a synapse?

The junction where one neuron communicates with another.

### Do artificial neurons perfectly model biological neurons?

No. They are simplified mathematical abstractions.
